This is how the heading will look in your WordPress site...

WordPress Implementation Instructions:

  1. Upload CSS: Upload the heading-styles.css file to your WordPress theme's CSS folder
  2. Enqueue the stylesheet: Add this to your theme's functions.php: 
    function enqueue_heading_styles() {
    wp_enqueue_style('custom-heading', get_template_directory_uri() . '/css/heading-styles.css');
}
add_action('wp_enqueue_scripts', 'enqueue_heading_styles');
  3. Use the class: In your WordPress editor, add the class "main-heading" to your heading: 
    <h1 class="main-heading">জনপ্রিয় ভিডিও</h1>
