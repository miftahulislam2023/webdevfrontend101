1. HTML Basics

What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It uses tags to structure content and tell browsers how to display information.

Key Concepts:
  • Tags are enclosed in angle brackets: <tagname>
  • Most tags have opening and closing pairs
  • Attributes provide additional information about tags
  • HTML documents have a specific structure

Basic HTML Document Structure

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>My First Website</title> </head> <body> <h1>Welcome to My Website</h1> <p>This is my first HTML page!</p> </body> </html>

Welcome to My Website

This is my first HTML page!

Every HTML document should start with this basic structure. It ensures proper rendering across all browsers and devices.

12. Accessibility

ARIA Attributes

<button aria-label="Close dialog" aria-expanded="false">√ó</button> <nav aria-label="Main navigation"> <ul role="menubar"> <li role="menuitem"><a href="#home">Home</a></li> <li role="menuitem"><a href="#about">About</a></li> </ul> </nav> <div role="alert" aria-live="polite"> Form submitted successfully! </div> <input type="password" aria-describedby="pwd-help"> <div id="pwd-help">Password must be at least 8 characters</div>
aria-label: Accessible name for screen readers
aria-describedby: References descriptive text
aria-live: Announces dynamic content changes

Semantic HTML for Accessibility

<main> <h1>Page Title</h1> <nav aria-label="Breadcrumb"> <ol> <li><a href="/">Home</a></li> <li><a href="/products">Products</a></li> <li aria-current="page">Current Page</li> </ol> </nav> <section> <h2>Section Heading</h2> <p>Content with proper heading hierarchy</p> </section> </main>
Proper semantic HTML provides structure that assistive technologies can understand and navigate.

Focus Management

<button tabindex="0">Focusable button</button> <div tabindex="-1">Programmatically focusable</div> <span tabindex="1">Higher tab order (avoid)</span> <a href="#content" class="skip-link">Skip to main content</a> <button onclick="document.getElementById('dialog').focus()"> Open Dialog </button>
tabindex="0": Makes element focusable in tab order
tabindex="-1": Programmatically focusable only
Skip links: Allow keyboard users to bypass navigation

‚ú® HTML Best Practices

Essential Guidelines:

  • Always use semantic HTML: Choose tags based on meaning, not appearance
  • Include alt text for images: Essential for accessibility
  • Use proper heading hierarchy: h1 ‚Üí h2 ‚Üí h3, don't skip levels
  • Validate your HTML: Use W3C validator to check for errors
  • Keep it simple: Use the least complex markup that works
  • Be consistent: Maintain consistent coding style
  • Comment your code: Especially for complex sections

Performance Tips:

  • Minimize HTTP requests by combining files
  • Use appropriate image formats and sizes
  • Implement lazy loading for images below the fold
  • Use CDNs for faster content delivery
  • Minify HTML in production
