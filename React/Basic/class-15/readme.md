# Class 15: JS in JSX Part 1

# কোথায় কার্লি ব্রেস ব্যবহার করবেন
- JSX-এর ভেতরে আপনি শুধুমাত্র **দুইভাবে** কার্লি ব্রেস ব্যবহার করতে পারেন:
  1. টেক্সট হিসেবে সরাসরি JSX ট্যাগের ভেতরে: `<h1>{name}'s To Do List</h1> `কাজ করে, কিন্তু <{tag}>MiftahCoding's To Do List</{tag}> কাজ করবে না।
  2. অ্যাট্রিবিউট হিসেবে সরাসরি = চিহ্নের পরে: src={avatar} লিখলে avatar ভেরিয়েবলটি পড়বে, কিন্তু src="{avatar}" লিখলে স্ট্রিং "{avatar}" পাস হবে।
